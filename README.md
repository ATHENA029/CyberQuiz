# CyberQuiz 🛡️

A cloud-based cybersecurity awareness training platform built on AWS.

## Features
- User registration and login via AWS Cognito
- Three interactive quiz modules (Phishing, Passwords, Social Engineering)
- Score tracking and digital badges
- Admin panel for monitoring user progress

## AWS Services Used
- Amazon S3 (hosting)
- AWS Lambda (backend logic)
- Amazon API Gateway (HTTP routes)
- Amazon DynamoDB (data storage)
- Amazon Cognito (authentication)
